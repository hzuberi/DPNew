## Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Design System

The website uses a colorful palette inspired by Melula:

- **Primary Colors**: Vibrant reds and pinks (#ff6b6b)
- **Secondary Colors**: Bright blues (#0ea5e9)
- **Accent Colors**: Warm yellows and oranges
- **Purple Accents**: Rich purples (#a855f7)

### Fonts

- **Display Font**: Poppins (for headings)
- **Body Font**: Inter (for body text)
